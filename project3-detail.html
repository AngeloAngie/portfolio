Windows server & DNS

Hier heb ik Windows Server geïnstalleerd. Tijdens het proces koos ik de gewenste versie, configureerde ik de taal- en netwerkopties en voltooide ik de installatie met succes. Windows Server biedt nu een stabiele basis voor netwerkbeheer, gebruikersbeheer en serverdiensten binnen mijn omgeving.

Hier heb ik DNS Manager geconfigureerd. Met DNS Manager kan ik domeinnaamresolutie beheren door DNS-zones en records aan te maken, zoals A-records en CNAME-records. Dit zorgt ervoor dat netwerkverkeer efficiënt wordt geleid en domeinnamen correct worden vertaald naar IP-adressen.

Hier heb ik het IPv4-adres geconfigureerd op de netwerkinterface. Door een statisch IP-adres in te stellen, zorg ik ervoor dat de server altijd hetzelfde adres gebruikt binnen het netwerk. Dit is belangrijk voor betrouwbare communicatie en toegang tot netwerkservices zoals DNS, DHCP en andere servers.
